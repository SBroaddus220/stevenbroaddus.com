FEH Game Dev. Project

Developed a video game in C++ using Ohio State's Proteus microcontroller simulator in a group with one other computer engineer. The simulator used to develop the game acted as a tool to draw shapes in a window, all logic was programmed manually for the game.

This game was designed to be a maze game where the user would drag a block using the mouse through a 2D maze. If the block touched any walls, it would reset the level. This project concluded in a competition where other peers would vote on which game was most fun, and this game won the "Best Game" award against 10 other honors engineering student groups.
